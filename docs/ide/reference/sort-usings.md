# <a name="sort-usings"></a>using 정렬

이 리팩터링은 다음에 적용됩니다.

- C#

**내용:** using 정렬

**시기:** 사전 순으로 표시되도록 파일의 맨 위에서 `using` 지시문을 정렬하려는 경우 

**이유:** using 지시문을 찾기 쉽게 합니다.

1. 메뉴 모음에서 **편집**을 선택합니다.
2. **Intellisense** > **Using 정렬**을 선택합니다.
