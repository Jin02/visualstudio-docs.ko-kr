# <a name="how-to-switch-between-views-and-the-xml-editor"></a>방법: 보기와 XML 편집기 간 전환

이 항목에서는 XML 스키마 디자이너 (XSD 디자이너) 뷰와 XML 편집기 간 전환 하는 방법을 보여 줍니다. 이 예제에서는 합니다 [구매 주문 스키마](../xml-tools/sample-xsd-file-simple-schema.md)합니다.

## <a name="to-switch-between-the-views-and-the-xml-editor"></a>뷰와 XML 편집기 간 전환

1. 를 만들고 새 XML 스키마 파일을 편집 하려면의 단계에 따라 [방법: XSD 스키마 파일 만들기 및 편집](../xml-tools/how-to-create-and-edit-an-xsd-schema-file.md)합니다.

2. 으로 전환 하려면 XML 스키마 디자이너는 XML 편집기에서 XML 편집기에서 아무 곳 이나 마우스 오른쪽 단추로 클릭 하 고 선택 **뷰 디자이너**합니다.

3. 워터 마크를 사용 하 여 그래프 뷰를 전환 하려면 클릭 합니다 **그래프 뷰를 사용 하 여 노드 간 관계를 볼** 시작 보기의 링크입니다.

4. 끌기 합니다 `USAddress` 에서 노드를 **XML 스키마 탐색기** 그래프 뷰로 합니다. 마우스 오른쪽 단추로 클릭 합니다 `USAddress` 그래프 뷰를 선택 하는 노드 **콘텐츠 모델 뷰로 표시** 상황에 맞는 메뉴입니다.

     `USAddress` 노드 정보가 있는 콘텐츠 모델 뷰가 나타납니다.

5. 도구 모음을 사용 하 여 콘텐츠 모델 뷰에서 시작 뷰로 전환 하려면 클릭 합니다 **시작 뷰** XSD 도구 모음 단추입니다.

6. 바로 가기 키를 사용 하 여 뷰 사이 전환 하려면 **Ctrl**+**1** 시작 보기의 경우 **Ctrl**+**2** 그래프 보기의 경우와 **Ctrl**+**3** 콘텐츠 모델 뷰에 대 한 합니다.

7. 콘텐츠 모델 뷰에서 XML 편집기 이동할 노드를 마우스 오른쪽 단추로 클릭 하 고 선택 **코드 보기** 상황에 맞는 메뉴입니다.
