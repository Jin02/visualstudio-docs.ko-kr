# <a name="color-theme-tools"></a>색상 테마 도구
Visual Studio SDK는 색상 테마를 만들고 관리하는 데 도움이 되는 몇 가지 도구를 제공합니다.

- [VSIX 컬러 편집기](../../extensibility/internals/vsix-color-editor.md) 도구는 Visual Studio용 사용자 지정 색상을 만들고 편집할 수 있습니다.

- [VSIX 컬러 컴파일러](../../extensibility/internals/vsix-color-compiler.md) 도구는 기존 Visual Studio 테마 *.xml* 파일을 확장자를 위한 *.pkgdef* 파일로 변환합니다.
