# <a name="outlining-and-hidden-text-in-a-legacy-language-service"></a>레거시 언어 서비스의 개요 표시 및 숨겨진 텍스트
개요를 사용하면 텍스트 줄의 시퀀스를 한 줄로 축소할 수 있습니다. 예를 들어 C++에서는 모든 메서드를 한 줄로 축소하여 메서드 시그니처만 표시할 수 있습니다. 숨겨진 텍스트는 표시되거나 숨길 수 있는 텍스트 줄의 시퀀스입니다.

 구현 방법 설명

 숨겨진 텍스트 영역의 용도를 설명하고 숨겨진 텍스트 영역을 구현하는 방법에 대한 지침을 제공합니다.

 정의 축소 명령을 지원하는 것 이상으로 언어에 대한 개요 지원을 확장하는 두 가지 옵션에 대해 *설명합니다.*
