# <a name="m_parent-field"></a>m_parent 필드

**네임스페이스:**<xref:System.Threading.Tasks?displayProperty=fullName>

**조립:** mscorlib *(mscorlib.dll)*

.NET Framework에서 이 내부 멤버에 액세스할 수 없으므로 CIL(일반 중간 언어)에서 다음 구문이 제공됩니다.

## <a name="syntax"></a>구문

```csharp

```
