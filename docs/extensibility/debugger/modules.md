# <a name="modules"></a>모듈
디버거 아키텍처의 *관점에서, 모듈*:

- 실행 파일 또는 DLL과 같은 코드의 실제 컨테이너입니다.

- 기호를 다시 로드하고 자신을 설명할 수 있습니다. 모듈 설명은 IDE의 모듈 창에 표시됩니다.

- 모듈을 설명하기 위해 디버그 엔진에 의해 생성된 [IDebugModule2](../../extensibility/debugger/reference/idebugmodule2.md) 인터페이스로 표시됩니다.
