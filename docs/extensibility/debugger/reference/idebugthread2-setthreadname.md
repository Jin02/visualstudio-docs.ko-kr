# <a name="idebugthread2setthreadname"></a>IDebugThread2::SetThreadName
스레드의 이름을 설정합니다.

## <a name="syntax"></a>구문

```cpp
HRESULT SetThreadName ( 
   LPCOLESTR pszName
);
```

```csharp
int SetThreadName ( 
   string pszName
);
```

## <a name="parameters"></a>매개 변수
`pszName`\
【인】 스레드의 이름입니다.

## <a name="return-value"></a>Return Value
 성공하면 반환합니다. `S_OK` 그렇지 않으면 오류 코드를 반환합니다.

## <a name="remarks"></a>설명
 스레드 이름을 얻으려면 [GetName](../../../extensibility/debugger/reference/idebugthread2-getname.md) 메서드를 호출합니다.
