# <a name="idebugport2getportid"></a>IDebugPort2::GetPortId
포트 식별자를 가져옵니다.

## <a name="syntax"></a>구문

```cpp
HRESULT GetPortId( 
   GUID* pguidPort
);
```

```csharp
int GetPortId( 
   out Guid pguidPort
);
```

## <a name="parameters"></a>매개 변수
`pguidPort`\
【아웃】 포트를 식별하는 GUID를 반환합니다.

## <a name="return-value"></a>Return Value
 성공하면 반환합니다. `S_OK` 그렇지 않으면 오류 코드를 반환합니다.
