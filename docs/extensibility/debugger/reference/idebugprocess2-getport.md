# <a name="idebugprocess2getport"></a>IDebugProcess2::GetPort
프로세스가 실행 중인 포트를 가져옵니다.

## <a name="syntax"></a>구문

```cpp
HRESULT GetPort( 
   IDebugPort2** ppPort
);
```

```csharp
int GetPort( 
   out IDebugPort2 ppPort
);
```

## <a name="parameters"></a>매개 변수
`ppPort`\
【아웃】 프로세스가 시작된 포트를 나타내는 [IDebugPort2](../../../extensibility/debugger/reference/idebugport2.md) 개체를 반환합니다.

## <a name="return-value"></a>Return Value
 성공하면 반환합니다. `S_OK` 그렇지 않으면 오류 코드를 반환합니다.
