# <a name="ienumdebugfieldsreset"></a>IEnumDebugFields::Reset
이 메서드는 열거형에서 첫 번째 요소로 다시 설정합니다.

## <a name="syntax"></a>구문

```cpp
HRESULT Reset(void);
```

```csharp
int Reset();
```

#### <a name="parameters"></a>매개 변수
 None

## <a name="return-value"></a>Return Value
 성공하면 반환합니다. `S_OK` 그렇지 않으면 오류 코드를 반환합니다.

## <a name="remarks"></a>설명
 이 메서드가 호출된 후 [Next에](../../../extensibility/debugger/reference/ienumdebugfields-next.md) 대한 다음 호출은 열거형의 첫 번째 요소를 반환합니다.
