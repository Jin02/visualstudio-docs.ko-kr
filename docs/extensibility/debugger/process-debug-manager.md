# <a name="process-debug-manager"></a>프로세스 디버그 관리자
PDM(프로세스 디버그 관리자)은 프로그램 및 프로세스를 관리하는 Visual Studio의 구성 요소로 세션 디버그 관리자와 디버그 엔진에서 사용할 수 있습니다.

 PDM은 디버깅할 수 있는 모든 프로세스를 관리합니다. 디버깅하려면 PDM에 프로그램을 등록해야 합니다. 이 등록은 프로그램이 시작될 때 포트 또는 디버그 엔진에 의해 수행됩니다.
