# <a name="debug-session"></a>디버그 세션
디버거 아키텍처의 관점에서 *세션*:

- IDE의 단일 인스턴스에서 시작되거나 연결된 관련 또는 관련없는 모든 프로세스의 집합입니다.

- 세션 디버그 관리자(SDM)에 의해 구현됩니다. 자세한 내용은 [세션 디버그 관리자](../../extensibility/debugger/session-debug-manager.md)를 참조하십시오.
