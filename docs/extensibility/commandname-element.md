# <a name="commandname-element"></a>CommandName 요소
`CommandName` 키보드 범주에 표시 되는 텍스트를 지정 하는 요소는 **옵션** 대화 상자에서 및를 **명령** 목록에 **사용자 지정** 대화 상자입니다.

## <a name="syntax"></a>구문

```
<CommandName>MyCommand</CommandName>
```
