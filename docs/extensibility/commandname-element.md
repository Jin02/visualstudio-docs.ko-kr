# <a name="commandname-element"></a>명령 이름 요소
이 `CommandName` 요소는 **옵션** 대화 상자의 키보드 범주에 나타나는 텍스트와 **사용자 지정** 대화 상자의 **명령** 목록에 있는 텍스트를 지정합니다.

## <a name="syntax"></a>구문

```
<CommandName>MyCommand</CommandName>
```
