# <a name="idiasymbolget_inlspec"></a>IDiaSymbol::get_InlSpec
이 함수는 [인라인, __inline, \__forceinline](/cpp/cpp/inline-functions-cpp) 특성 중 하나를 사용 하 여 함수가 인라인으로 표시 되었는지 여부를 나타내는 플래그를 검색 합니다.

## <a name="syntax"></a>구문

```C++
HRESULT get_inlSpec(
   BOOL *pRetVal
);
```

#### <a name="parameters"></a>매개 변수
 `pRetVal`

제한이 함수가 인라인으로 표시 되 면 `TRUE`을 반환 합니다. 그렇지 않으면 `FALSE`을 반환 합니다.

## <a name="return-value"></a>반환 값
 성공 하면 `S_OK`을 반환 합니다. 그렇지 않으면 `S_FALSE` 또는 오류 코드를 반환 합니다.

> [!NOTE]
> @No__t_0의 반환 값은 해당 속성을 기호에 사용할 수 없음을 의미 합니다.
