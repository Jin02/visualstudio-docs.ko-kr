# <a name="idiasymbolget_hassetjump"></a>IDiaSymbol::get_hasSetJump
함수에 [setjmp](/cpp/c-runtime-library/reference/setjmp) 명령이 사용 되는지 여부를 지정 하는 플래그를 검색 합니다. 여기서는 [cjmp](/cpp/c-runtime-library/reference/longjmp) 명령과 쌍을 이룹니다. 이러한 명령은 예외 처리의 C 스타일 메서드를 형성 합니다.

## <a name="syntax"></a>구문

```C++
HRESULT get_hasSetJump(
   BOOL *pFlag
);
```

#### <a name="parameters"></a>매개 변수
 `pFlag`

제한이 함수에 `setjmp` 명령이 포함 된 경우 `TRUE`를 반환 합니다. 그렇지 않으면 `FALSE`을 반환 합니다.

## <a name="return-value"></a>반환 값
 성공 하면 `S_OK`을 반환 합니다. 그렇지 않으면 `S_FALSE` 또는 오류 코드를 반환 합니다.

> [!NOTE]
> @No__t_0의 반환 값은 해당 기호에 대해 속성을 사용할 수 없음을 의미 합니다.
