# <a name="idiaenumtablesgetcount"></a>IDiaEnumTables::get_Count
테이블을 검색합니다.

## <a name="syntax"></a>구문

```C++
HRESULT get_Count (    LONG* pRetVal
);

```

#### <a name="parameters"></a>매개 변수
 `pRetVal`

[out] 테이블의 수를 반환합니다.

## <a name="return-value"></a>반환 값
 성공 하면 반환 `S_OK`고, 그렇지 않으면 오류 코드를 반환 합니다.
