# <a name="idiaenumlinenumbersreset"></a>IDiaEnumLineNumbers::Reset

## <a name="syntax"></a>구문

```C++
HRESULT Reset();
```

## <a name="return-value"></a>반환 값
 `S_OK`을 반환합니다.
