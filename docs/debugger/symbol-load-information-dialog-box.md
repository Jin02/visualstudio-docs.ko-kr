# <a name="symbol-load-information-dialog-box"></a>기호 로드 정보 대화 상자
**기호 로드 정보** 대화 상자는 **모듈** 창을 마우스 오른쪽 단추로 클릭하고 **기호 로드 정보**를 선택하면 열립니다.

## <a name="uielement-list"></a>UI 요소 목록
 **정보** Visual Studio 설치 설정에 따라 기호 파일 (PDB 파일)을 검색 한 디렉터리와 **옵션** 대화 상자에서 기호 설정에 대 한 변경 내용을 보여 줍니다.

 **기호 설정** **옵션** 대화 상자 (**디버깅** 범주, **기호** 페이지)에서 기호 설정을 변경 하려면이 단추를 클릭 합니다. 이 페이지에서 기호 위치를 추가하거나 제거하고, 기호 서버를 사용하는 경우 로컬 캐시 위치를 지정하고, 기호 위치를 검색할 조건을 제어할 수 있습니다.
