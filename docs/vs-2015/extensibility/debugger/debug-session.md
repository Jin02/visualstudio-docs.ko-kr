[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

디버거 아키텍처 측면을 **세션**:  
  
-   관련 또는 관련이 없는 모든 집합 처리를 시작 또는 IDE의 단일 인스턴스에서 연결 된 경우
