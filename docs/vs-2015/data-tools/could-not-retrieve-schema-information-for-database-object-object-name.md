[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

경우에 개체에 일반적으로이 메시지가 나타납니다 **서버 탐색기/데이터베이스 탐색기** 클립보드에 복사, 삭제, 데이터베이스에서 되어 디자이너에 붙여넣은 후 합니다. 즉, 데이터베이스 개체가 더 이상 존재하지 않으므로 이 메시지가 나타납니다.
