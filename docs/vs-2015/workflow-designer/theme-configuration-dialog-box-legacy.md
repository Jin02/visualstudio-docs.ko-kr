# <a name="theme-configuration-dialog-box-legacy"></a>테마 구성 대화 상자(레거시)
에 대해 설명 하는 방법을 사용 하 여는 **테마 구성** 레거시 대화 상자 [!INCLUDE[wfd1](../includes/wfd1-md.md)]합니다. 레거시 [!INCLUDE[wfd2](../includes/wfd2-md.md)]는 [!INCLUDE[netfx35_long](../includes/netfx35-long-md.md)] 또는 [!INCLUDE[vstecwinfx](../includes/vstecwinfx-md.md)]를 대상으로 해야 하는 경우에 사용합니다.  
  
 테마는 워크플로의 배경색 및 전경색, 스타일, 아이콘 및 기타 시각적 요소를 정의합니다. 테마를 저장했다가 다른 워크플로에서 다시 사용할 수 있습니다.  
  
 만들고 사용 하 여 테마를 편집 합니다 **테마 구성** 대화 상자. 선택 대화 상자를 열려면 **새 테마 만들기** 에 **워크플로** 메뉴나 워크플로 디자인 화면을 마우스 오른쪽 단추로 클릭 **새 테마 만들기**합니다.  
  
 다음 표에 사용자 인터페이스 (UI) 요소에는 **테마 구성** 대화 상자.  
  
|UI 요소|설명|  
|----------------|-----------------|  
|**테마 이름:**|테마를 식별 하는 이름 합니다 [테마, 워크플로 디자이너, 옵션 대화 상자 (레거시)](../workflow-designer/themes-workflow-designer-options-dialog-box-legacy.md)합니다. 새 테마의 이름이 생성되며 이 이름은 변경 가능합니다.|  
|**테마 위치:**|테마 파일의 파일 이름 및 경로입니다. 생성된 테마 이름에 따라 새 테마의 파일 이름이 생성되며 이 이름은 변경 가능합니다. 생성된 테마 이름을 변경하면 파일 이름도 테마 이름과 일치하도록 변경할 수 있습니다.|  
|**...**|워크플로 테마 파일을 저장할 위치를 선택하려면 클릭합니다. 이 파일의 확장명은 .wtm입니다. 선택한 경로에 표시 되는 **테마 위치** 입력란입니다.|  
|**디자이너를 선택 하 고 속성을 구성 합니다.**|왼쪽 창에는 테마를 사용자 지정할 수 있는 해당 활동의 트리 뷰가 나타납니다. 트리 뷰에서 활동을 선택하면 그 활동의 테마 속성이 트리 뷰 창 오른쪽의 속성 창에 표시됩니다. 값을 변경할 속성을 클릭합니다.|  
|**미리 보기**|속성 변경 결과를 미리 보기 위해 창을 표시하려면 클릭합니다.|
