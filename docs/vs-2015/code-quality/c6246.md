# <a name="c6246"></a>C6246
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

경고 C6246: 지역 선언이 \<변수 > 외부 범위에 있는 같은 이름의 선언을 숨깁니다. 추가 정보: 에 있는 이전 선언을 참조 \<위치 >.  
  
 이 경고는 로컬 범위에서 선언 두 가지 이름이 있는지를 나타냅니다. 외부 범위에서 이름은 내부 범위에서 선언에 의해 숨겨집니다. 로컬 선언 사용 하는 외부 범위 선언의 모든 용도 발생 합니다.  
  
## <a name="example"></a>예제  
 다음 코드에서는이 경고를 생성합니다.  
  
```  
#include <stdlib.h>  
#define UPPER_LIMIT 256  
int DoSomething( );  
  
int f( )  
{  
  int i = DoSomething( );  
  if (i > UPPER_LIMIT)  
  {  
    int i;  
    i = rand( );  
  }  
  return i;  
}  
```  
  
 이 경고를 해결 하려면 다음 코드에 표시 된 것과 같이 다른 변수 이름을 사용 합니다.  
  
```  
#include <stdlib.h>  
#define UPPER_LIMIT 256  
int DoSomething( );  
  
int f ( )  
{  
  int i = DoSomething( );  
  if (i > UPPER_LIMIT)  
  {  
    int j = rand( );  
    return j;  
  }  
  else  
  {  
    return i;  
  }  
}  
```  
  
 이 경고는만 범위 중복을 식별합니다.
