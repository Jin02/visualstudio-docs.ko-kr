# <a name="ca0505"></a>CA0505

Superceding 속성이 정의 되어 있기 때문에 사용 되지 않는 속성이 무시 됩니다.

지정 된 사용 되지 않는 및 superceding 코드 분석 속성 모두 프로젝트 또는 빌드 구성 파일의 값을 포함 하는 경우 경고 CA505 생성 됩니다. 이 문제를 해결 하려면 사용 되지 않는 속성을 제거 합니다.
