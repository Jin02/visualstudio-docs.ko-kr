# <a name="c28280"></a>C28280
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

경고 C28280: 기호의 경우 일치하는 'begin'이 없는 'end'가 있습니다.  
  
 주석 언어가 begin 및 end를 지원 합니다 (`{` 및 `}` c에서) 구성 하 고 쌍의 균형이 합니다. 매크로를 사용하는 경우 이 상황을 방지할 수 있습니다.
