# <a name="c28020"></a>C28020
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

경고 C28020:이 호출에서 식 \<expr >이 true가 아닙니다.  
  
 이 경고는 나열 된\_ 식을 충족 하는 \_이 true가 아닌 경우에 보고 됩니다. 이 경고는 종종 잘못된 매개 변수를 나타냅니다.  
  
 함수 선언에서 이 경고가 발생할 경우 주석은 불가능한 상태를 나타냅니다.
