# <a name="c6504"></a>C6504
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

경고 C6504: 주석이 잘못 되었습니다: 속성 포인터, 멤버에 대 한 포인터 또는 배열 형식의 값에만 사용할 수 있습니다  
  
 이 경고는 호환 되지 않는 데이터 형식에서 속성의 사용을 나타냅니다. 속성에서 지 원하는 데이터 형식에 대 한 자세한 내용은 참조 하세요. [주석 속성](http://msdn.microsoft.com/f77b4370-6bda-4294-bd2a-e7d0df182a3d)합니다.  
  
## <a name="example"></a>예제  
 다음 코드에서는 `_Null_` 속성을 참조 데이터 형식에 사용할 수 없기 때문에 이 경고가 발생합니다.  
  
```cpp  
#include<sal.h>  
  
class Point  
{  
public:  
    //  members  
};  
  
void f(_Pre_ _Null_ Point& pt)  
{  
    // code ...  
}  
  
```  
  
 이 경고를 해결 하려면 다음 코드를 사용 합니다.  
  
```cpp  
#include<sal.h>  
  
class Point  
{  
public:  
    //  members  
};  
  
void f(_Pre_ _Null_  Point* pt)  
{  
    // code ...  
}  
```  
  
 앞의 코드 생성 경고 [C6516](../code-quality/c6516.md) 때문 속성 충돌에는 주석이 잘못 되었습니다.
