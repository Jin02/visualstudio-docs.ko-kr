# <a name="c6506"></a>C6506
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

경고 C6506: 주석이 잘못 되었습니다: \<이름 > 속성 포인터 또는 배열 형식의 값에만 사용할 수 있습니다  
  
 이 경고 속성 포인터 또는 배열 형식 이외의 형식에 사용 됨을 나타냅니다. 액세스, Tainted, 및 유효한 속성이 모든 데이터 형식에 수입니다. ValidBytesConst, ValidElementsConst, ElementSize, NullTerminted 등의 다른 속성에 대 한 포인터, 멤버 또는 배열 형식에 대 한 포인터를 지원합니다. 속성 및 지원 되는 데이터 형식의 전체 목록은 참조 하세요 [주석 속성](http://msdn.microsoft.com/f77b4370-6bda-4294-bd2a-e7d0df182a3d)합니다.  
  
## <a name="example"></a>예제  
 다음 코드에서는이 경고를 생성합니다.  
  
```cpp  
#include<sal.h>  
void f(_Out_ char c)  
{  
    c = 'd';  
}  
```  
  
 이 경고를 해결 하려면 다음 샘플 코드와 같이 포인터 또는 배열 형식을 사용 합니다.  
  
```cpp  
#include<sal.h>  
void f(_Out_ char *c)  
{  
    *c = 'd';  
}  
```
