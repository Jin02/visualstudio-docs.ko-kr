# <a name="ca0057"></a>CA0057

위반 대상 및 규칙의 현재 집합에 매핑할 수 없습니다.

FxCopCmd.exe 프로젝트 파일을 사용 하 여 수동으로 호출 하는 경우이 오류가 발생 또는 / 대상 및 규칙의 현재 집합에 매핑할 수 없는 위반 있는 XML 보고서를 가져옵니다.

분석을 수행 하는 경우에 발생 도구의 명령줄 버전을 사용 합니다.
