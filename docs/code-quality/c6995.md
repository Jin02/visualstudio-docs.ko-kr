# <a name="c6995"></a>C6995
경고 C6995: XML 로그 파일을 저장하지 못했습니다.

 이 경고는 코드 분석 도구가 코드 분석의 출력인 오류 로그를 만들 수 없음을 나타냅니다.

 이 오류는 디스크 오류를 나타내거나 지정된 디렉터리에 파일을 만들 수 있는 권한이 없음을 나타낼 수 있습니다.
