# <a name="c28246"></a>C28246
경고 C28246: 함수 '\<name > '-parameter '\<매개 변수 > '에 대 한 주석이 매개 변수의 형식과 일치 하지 않습니다.

 주석을 만들 때 __deref 연산자가 포인터가 아닌 형식에 적용되었습니다.
