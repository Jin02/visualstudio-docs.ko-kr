# <a name="c6272"></a>C6272
경고 C6272: \<function을 호출 하는 데 float가 필요한 경우 부동 소수점 @no__t 인수로 전달 되는-0number > >

 이 경고는 형식 문자열에서 float가 필요 함을 나타냅니다. 예를 들어 `printf,`에 대 한 `%f` 또는 `%g` 지정 이지만 정수 또는 문자열과 같이 부동 소수점 형식이 전달 되지 않음을 나타냅니다. 이 결함으로 인해 잘못 된 출력이 발생할 수 있습니다. 그러나 특정 상황에서는 충돌이 발생할 수 있습니다.

## <a name="example"></a>예제
 다음 코드에서는이 경고를 생성 합니다.

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf(buff,"%s %f","a",i);
}
```

 이 경고를 해결 하려면 다음 코드와 같이 `%f` 지정 대신 `%i`을 사용 합니다.

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf(buff,"%s %i","a",i);
}
```

 다음 코드에서는 safe 문자열 조작 @no__t 함수를 사용 하 여이 경고를 수정 합니다.

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf_s(buff,5,"%s %i","a",i); // safe version
}
```
