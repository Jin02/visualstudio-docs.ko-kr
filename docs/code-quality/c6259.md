# <a name="c6259"></a>C6259
경고 C6259: 레이블이 지정 된 코드에 접근할 수 없습니다. (\<expression > & \< 상수 >).

 이 경고는 스위치 식에서 비트 and (`&`) 비교 결과로 발생 하는 접근할 수 없는 코드를 나타냅니다. 스위치 식의 상수와 일치 하는 case 문에만 접근할 수 있습니다. 다른 모든 case 문에 연결할 수 없습니다.

## <a name="example"></a>예제
 다음 샘플 코드에서는 `switch``(rand() & 3)` 식을 case 레이블 (`case` `4`)로 평가할 수 없기 때문에이 경고를 생성 합니다.

```
#include <stdlib.h>

void f()
{
     switch (rand () & 3) {
        case 3:
            /* Reachable */
            break;
        case 4:
            /* Not reachable */
            break;
        default:
            break;
    }
}
```

 이 경고를 해결 하려면 접근할 수 없는 코드를 제거 하거나 case 문에 사용 된 상수가 올바른지 확인 하십시오. 다음 코드는 접근할 수 없는 case 문을 제거 합니다.

```
#include <stdlib.h>

void f()
{
     switch (rand () & 3) {
        case 3:
            /* Reachable */
            break;
        default:
            break;
    }
}
```
