# <a name="c28275"></a>C28275
경고 C28275: @No__t-0 매크로 @ no__t-1value @ no__t-2에 대 한 매개 변수가 null입니다.

 이 경고는 분석 중인 코드가 아니라 모델 파일에 내부 오류가 있음을 나타냅니다. *MacroValue* 함수가 매개 변수 없이 호출 되었습니다.
