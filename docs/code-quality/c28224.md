# <a name="c28224"></a>C28224
경고 C28224: 주석에 매개 변수가 필요 합니다.

 이 경고는 명명 된 주석이 매개 변수 없이 사용 되 고 하나 이상의 매개 변수가 필요 함을 나타냅니다. 주석에 표준 매크로를 사용하는 경우에는 이 경고가 발생할 수 없습니다. 일반적인 사용 시에는 이 경고가 발생하지 않습니다.
