# <a name="c28210"></a>C28210
경고 28210: @No__t-0 컨텍스트에 대 한 주석이 명시적 사전 컨텍스트에 없어야 합니다.

 `_On_failure_` 주석은 `__post` 컨텍스트에서 명시적 또는 암시적으로 표시되어야 합니다. 즉, 함수 반환 후에 적용되어야 합니다.  이렇게 하려면 `_drv_out`을 사용합니다.
