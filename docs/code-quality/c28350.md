# <a name="c28350"></a>C28350
경고 C28350: 주석 \<annotation >는 조건부로 적용할 수 없는 상황을 설명 합니다.

 일반적으로 이 경고는 C/C++ 형식을 검사 중인 위치에 주석이 적용될 때 생성됩니다.
