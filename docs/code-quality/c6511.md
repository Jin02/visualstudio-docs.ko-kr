# <a name="c6511"></a>C6511
경고 C6511: 주석이 잘못 되었습니다. MustCheck 속성 이어야 합니다 예 또는 아니요

> [!NOTE]
> 이 경고는 사용되지 않는 버전의 SAL(소스 코드 주석 언어)을 사용하는 코드에서만 발생합니다. 최신 버전의 SAL을 사용하기 위해 코드를 이식하는 것이 좋습니다. 자세한 내용은 [Reduce C SAL 주석을 사용 하 여 /C++ 코드 결함](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md)합니다.

 이 경고는 잘못 되었습니다. MustCheck 속성 값이 지정을 나타냅니다. 이 속성에만 유효한 값은: 그렇기도 하고 그렇지 않기도 하고.

## <a name="example"></a>예제
 다음 코드에서는이 경고를 생성합니다.

```
// C
#include <CodeAnalysis\SourceAnnotations.h>
[returnvalue:SA_Post(MustCheck=SA_Maybe)] int f();

// C++
#include <CodeAnalysis\SourceAnnotations.h>
using namespace vc_attributes;
[returnvalue:Post(MustCheck=Maybe)] int f();
```

 이 경고를 해결 하려면 MustCheck 속성에 대 한 유효한 값은 다음 코드에 사용 됩니다.

```
// C
#include <CodeAnalysis\SourceAnnotations.h>
[returnvalue:SA_Post(MustCheck=SA_Yes)] int f();

// C++
#include <CodeAnalysis\SourceAnnotations.h>
using namespace vc_attributes;
[returnvalue:Post(MustCheck=Yes)] int f();
```
