# <a name="c1255"></a>C1255
경고 C1255: 플러그 인에 대 한 PCH 데이터의 길이가 잘못 되었습니다.

 코드 분석 도구는 분석 중인 코드가 아니라 도구에 내부 오류가 있을 때 이 경고를 보고합니다.
