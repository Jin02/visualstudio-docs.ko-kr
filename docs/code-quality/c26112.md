# <a name="c26112"></a>C26112
경고 C26112: @No__t-0func >를 호출 하기 전에 호출자가 잠금을 유지할 수 없습니다.

 주석 `_Requires_no_locks_held_`은 호출자가 함수를 호출 하는 동안 잠금을 유지 하지 않아야 하는 사전 조건을 적용 합니다. 경고 C26112는 다른 함수를 호출 하기 전에 함수가 모든 잠금을 해제 하는 데 실패 한 경우에 발생 합니다.

## <a name="example"></a>예제
 다음 예에서는 잠긴 섹션 내에서 `NoLocksAllowed`을 호출 하 여 `_Requires_no_locks_held_` 사전 조건을 위반 했기 때문에 경고 C26112을 생성 합니다.

```

typedef struct _DATA
{
    CRITICAL_SECTION cs;
} DATA;

_Requires_no_locks_held_

void NoLocksAllowed(DATA* p)
{
     // Lock sensitive operations here
}

void LocksHeldFunction(DATA* p)
{
    EnterCriticalSection(&p->cs);
    NoLocksAllowed(p); // Warning C26112
    LeaveCriticalSection(&p->cs);
}
```
