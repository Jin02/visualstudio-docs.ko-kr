# <a name="c26466-nostaticdowncastpolymorphic"></a>C26466 NO_STATIC_DOWNCAST_POLYMORPHIC
  사용 하지 않는 `static_cast` 다운 캐스트 합니다. 다형 형식에서 캐스팅은 dynamic_cast을 사용 해야 합니다. 참조 [C++ 핵심 지침 Type.2](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Pro-type-downcast)합니다.
