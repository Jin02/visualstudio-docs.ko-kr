# <a name="c26466-no_static_downcast_polymorphic"></a>C26466 NO_STATIC_DOWNCAST_POLYMORPHIC
  `static_cast` downcasts를 사용 하지 마세요. 다형 형식에서 캐스팅은 dynamic_cast을 사용 해야 합니다. [ C++ 핵심 지침 유형 2](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Pro-type-downcast)를 참조 하세요.
