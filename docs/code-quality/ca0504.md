# <a name="ca0504"></a>CA0504
규칙 디렉터리가 없으므로 무시 되었습니다.

경고 CA504는 프로젝트 또는 빌드 구성 파일의 **CodeAnalysisRuleDirectories** 속성에 지정 된 디렉터리를 찾을 수 없는 경우에 생성 됩니다. 이 문제를 해결 하려면 올바른 디렉터리 경로를 입력 하십시오.
