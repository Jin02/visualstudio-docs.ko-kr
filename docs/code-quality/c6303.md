# <a name="c6303"></a>C6303
경고 C6303: 형식 문자열이 일치 하지 않습니다. \<function에 대 한 호출에 문자열이 필요한 경우 매개 변수로 전달 된 와이드 문자열 \<number > >

 이 경고는 형식 문자열에서 문자열이 필요 함을 지정 했음을 나타냅니다. 그러나 와이드 문자열을 전달 하 고 있습니다. 이 결함으로 인해 일부 형태의 크래시 또는 손상이 발생할 수 있습니다.

## <a name="example"></a>예제
 다음 샘플 코드에서는이 경고를 생성 합니다.

```
#include <stdio.h>

void f()
{
  wchar_t buff[5] = L"hi";

  printf("%s", buff);
}
```

 이 경고를 해결 하려면 다음 샘플 코드에 표시 된 대로 `%ls`를 사용 합니다.

```
#include <stdio.h>

void f()
{
  wchar_t buff[5] = L"hi";

  printf("%ls", buff);
}
```

 다음 샘플 코드는 `printf_s` 안전한 문자열 조작 함수를 사용 하 여이 경고를 수정 합니다.

```
#include <stdio.h>

void f()
{
  wchar_t buff[5] = L"hi";

  printf_s("%ls",buff);
}
```
