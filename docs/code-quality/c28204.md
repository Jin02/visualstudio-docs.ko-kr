# <a name="c28204"></a>C28204
경고 C28204: \<function >:이 오버 로드 중 하나 및 \<filename > (\<line >)에는 \<paramname >에 대 한 주석이 추가 됩니다. 또는 둘 다에 주석을 추가 해야 합니다.

 이 경고는 주석에 오류가 있는 경우 보고됩니다.
