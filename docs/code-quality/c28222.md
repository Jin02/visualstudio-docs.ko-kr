# <a name="c28222"></a>C28222
경고 28222: 주석에는 \_Yes \_, \_No \_ 또는 \_Maybe \_ 필요 합니다.

 이 경고는 주석의 매개 변수로 `_Yes_`, `_No_` 또는 `_Maybe_` 기호 중 하나가 필요한데 다른 기호가 발견되었음을 나타냅니다. 일반적으로 이 경고는 잘못 코딩된 주석 매크로를 나타냅니다.
