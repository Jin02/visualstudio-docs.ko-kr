# <a name="c28219"></a>C28219
경고 C28219: 주석에 있는 매개 변수 주석에 열거의 멤버가 필요합니다.

 주석에 대 한 매개 변수는 명명 된 `enum` 형식의 멤버 여야 하는데 일부 다른 기호가 있습니다. `enum` 유형의 멤버를 사용 합니다. 일반적으로 이 경고는 잘못 코딩된 주석 매크로를 나타냅니다.
