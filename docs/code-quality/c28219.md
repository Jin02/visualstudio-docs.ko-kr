# <a name="c28219"></a>C28219
warning C28219: 주석에 있는 매개 변수 주석에 열거의 멤버가 필요합니다.

 주석의 매개 변수 명명 된 멤버 이어야 하는데 `enum` 형식 및 다른 기호가 발견 되었습니다; 멤버를 사용 하 여 `enum` 형식입니다. 일반적으로 이 경고는 잘못 코딩된 주석 매크로를 나타냅니다.
