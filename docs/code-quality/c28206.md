# <a name="c28206"></a>C28206
경고 C28206: \<식 >: 왼쪽 피연산자가 구조체를 가리키고 `->`를 사용 하십시오.

 이 경고는 구조체 포인터 역참조 표기법 `->`가 필요한 경우 보고됩니다.
