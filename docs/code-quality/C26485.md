C26481, 같은이 확인란을이 선택 하면 I.13 규칙을 적용 합니다. *단일 포인터 배열을 전달 하지 마십시오* 정적 배열 형식 정보를 원시 포인터로 인해 고갈 상태가 손실 가져옵니다 있는 위치를 검색 합니다. zstring 형식과 czstring 제외 되지 않습니다.
