# <a name="c26167"></a>C26167
경고 C26167: 함수 \<func >에서 유지 되지 않은 잠금 \<lock >를 해제할 수 있습니다.

 경고 C26167는 신뢰도 수준이 낮은 것을 제외 하 고 경고 [C26117](../code-quality/c26117.md) 유사 합니다. 예를 들어 함수에 주석 오류가 포함될 수 있습니다.

## <a name="example"></a>예제
 다음 코드는 C26167뿐만 아니라 C26110도 생성합니다.

```cpp

typedef struct _DATA {
    CRITICAL_SECTION cs;
} DATA;

_Releases_lock_(p->cs) void Leave(DATA* p) {
    LeaveCriticalSection(&p->cs); // OK
}
void ReleaseUnheldLock(DATA* p) { // Warning C26167
    int i = 0;
    Leave(p); // Warning C26110
}
```

## <a name="example"></a>예제
 다음 코드에서는 이러한 경고가 해결됩니다.

```cpp

typedef struct _DATA {
    CRITICAL_SECTION cs;
} DATA;

_Releases_lock_(p->cs) void Leave(DATA* p) {
    LeaveCriticalSection( &p->cs );
}

void ReleaseUnheldLock( DATA* p ) {
    EnterCriticalSection( &p->cs );
    int i = 0;
    Leave(p);
}
```
