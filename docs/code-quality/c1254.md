# <a name="c1254"></a>C1254
경고 C1254: 플러그 인 버전 불일치: 버전이 PREfast 드라이버의 버전과 일치 하지 않습니다.

 코드 분석 도구는 분석 중인 코드가 아니라 플러그 인 버전에 내부 오류가 있는 경우에 이 경고를 보고합니다.
