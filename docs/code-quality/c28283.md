# <a name="c28283"></a>C28283
경고 C28283: 기호의 경우 지정 된 크기 사양이 아직 지원 되지 않습니다.

 이 경고는 크기 사양 "센티널" 주석이 0이 아닌 값을 받았음을 나타냅니다. 기본적으로 호출자는 문자열이 이진 0 이외의 문자로 끝남을 나타내려고 했습니다.
