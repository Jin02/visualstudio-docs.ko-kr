# <a name="c26497-use_constexpr_for_function"></a>C26497 USE_CONSTEXPR_FOR_FUNCTION

컴파일 시간 계산이 필요한 경우% function% 함수를 `constexpr`으로 표시할 수 있습니다. 참조 [c + + Core Guidelines F.4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rf-constexpr)합니다.
