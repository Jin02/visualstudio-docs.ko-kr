# <a name="c28234"></a>C28234
경고 C28234: \_At \_ 식이 현재 함수에 적용 되지 않습니다.

 이 경고는 `_At_` 식의 값이 액세스 가능한 개체를 식별하지 않음을 나타냅니다.
