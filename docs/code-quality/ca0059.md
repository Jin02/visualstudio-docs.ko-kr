# <a name="ca0059"></a>CA0059
명령줄 전환 오류입니다.

이 오류는 필요한 명령줄 스위치가 없는 경우에 발생 합니다. 예를 들면 다음과 같습니다.

/out 또는/console 스위치가 지정 되지 않았습니다. 잘못 된 시간 제한 값과 같은 잘못 된 스위치가 있는 경우에도이 오류가 발생 합니다.
