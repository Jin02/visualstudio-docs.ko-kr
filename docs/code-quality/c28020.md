# <a name="c28020"></a>C28020
경고 C28020:이 호출에서 식 \<expr >이 (가) true가 아닙니다.

 이 경고는 나열 된 \_Satisfies \_ 식이 true가 아닐 때 보고 됩니다. 이 경고는 종종 잘못된 매개 변수를 나타냅니다.

 함수 선언에서 이 경고가 발생할 경우 주석은 불가능한 상태를 나타냅니다.
