# <a name="c26471-no_reinterpret_cast_from_void_ptr"></a>C26471 NO_REINTERPRET_CAST_FROM_VOID_PTR

`reinterpret_cast`사용 하지 마세요. Void *에서 캐스트는 `static_cast`사용할 수 있습니다. [ C++ 핵심 지침 형식. 1](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Pro-type-reinterpretcast)을 참조 하세요.
