# <a name="c26471-noreinterpretcastfromvoidptr"></a>C26471 NO_REINTERPRET_CAST_FROM_VOID_PTR

사용 하지 않는 `reinterpret_cast`합니다. Void *에서 캐스트를 사용 하 여 수 `static_cast`입니다. 참조 [C++ 핵심 지침 Type.1](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Pro-type-reinterpretcast)합니다.
