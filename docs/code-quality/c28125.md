# <a name="c28125"></a>C28125
경고 C28125: 함수는 try/except 블록 내에서 호출 해야 합니다.

 드라이버에서 [ProbeForRead](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-probeforread), [ProbeForWrite](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-probeforwrite), [MmProbeAndLockPages](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-mmprobeandlockpages)등의 try/except 블록 내에서 호출 해야 하는 함수를 호출 하 고 있습니다.

## <a name="example"></a>예제
 다음 코드 예제에서는 이 경고를 생성합니다.

```
ProbeForRead(addr, len, 4);
```

 다음 코드 예제에서는 이 경고를 방지합니다.

```

__try
{
    ProbeForRead(addr, len, 4);
}
__except(EXCEPTION_EXECUTE_HANDLER)
{
    Status = GetExceptionCode();
    ... report error status
}
```
