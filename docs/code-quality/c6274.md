# <a name="c6274"></a>C6274
경고 C6274: \<function를 호출 하는 데 문자가 필요한 경우 \<number > 매개 변수로 전달 되지 않습니다 >

 이 경고는 형식 문자열에서 문자가 필요 하다는 것을 지정 합니다 (예: `%c` 또는 `%C` 지정). 그러나 float, string 또는 struct와 같은 정수가 아닌 값이 전달 되 고 있음을 나타냅니다. 이 결함으로 인해 잘못 된 출력이 발생할 수 있습니다.

## <a name="example"></a>예제
 다음 코드에서는이 경고를 생성 합니다.

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf(buff,"%c",str);
}
```

 이 경고를 해결 하려면 다음 코드를 사용 합니다.

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf(buff,"%c",str[0]);
}
```

 다음 코드에서는 `sprintf_s`을 사용 하 여이 경고를 해결 하는 안전한 문자열 조작 함수를 사용 합니다.

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf_s(buff,5,"%c", str[0]);
}
```
