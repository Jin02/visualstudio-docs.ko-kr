# <a name="c6274"></a>C6274
경고 C6274: \<function 호출에 문자가 필요한 경우 > \<number 매개 변수로 전달 되는 문자가 아닙니다 >

 이 경고는 형식 문자열이 문자 (예: `%c` 또는 `%C` 사양)를 지정 하지만 float, string 또는 struct와 같이 정수가 아닌 정수를 전달 하 고 있음을 나타냅니다. 이 결함으로 인해 잘못 된 출력이 발생할 수 있습니다.

## <a name="example"></a>예제
 다음 코드에서는이 경고를 생성 합니다.

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf(buff,"%c",str);
}
```

 이 경고를 해결 하려면 다음 코드를 사용 합니다.

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf(buff,"%c",str[0]);
}
```

 다음 코드는 `sprintf_s` 안전한 문자열 조작 함수를 사용 하 여이 경고를 해결 합니다.

```
#include <stdio.h>
#include <string.h>

void f(char str[])
{
   char buff[5];

   sprintf_s(buff,5,"%c", str[0]);
}
```
