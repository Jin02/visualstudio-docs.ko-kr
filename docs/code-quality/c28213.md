# <a name="c28213"></a>C28213
warning C28213: `_Use_decl_annotations_` 이전 선언을 수정 하지 않고 참조 하려면 주석을 사용 해야 합니다. \<why>

 이 경고는 필요한 `_Use_decl_annotations_` 주석 없이 이전 선언이 참조되었을 때 보고됩니다.
