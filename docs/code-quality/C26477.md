# <a name="c26477-usenullptrnotconstant"></a>C26477 USE_NULLPTR_NOT_CONSTANT

`nullptr` 특수 형식이 `nullptr_t` 특별 한 null 처리를 사용 하 여 오버 로드를 허용 하는 합니다. 사용 하 여 `0` 나 `NULL` 대신 `nullptr` 형식 안전성 및 추론 무시는 `nullptr` 제공 합니다.
