# <a name="c6317"></a>C6317
경고 C6317: 잘못 된 연산자: 논리 부정 (!) (~) 보수와 아닙니다.

 이 경고에 논리 부정 (`!`) 상수는 비트 플래그 여야 가능성이 있는 적용 되 고 있습니다. 논리 부정 결과 부울입니다. 비트 적용할 올바르지-및 (`&`) 연산자는 부울 값입니다. 보수를 사용 하 여 (`~`) 연산자를 플래그를 대칭 이동 합니다.

## <a name="example"></a>예제
 다음 코드에서는이 경고를 생성합니다.

```
#define FLAGS   0x4004

void f(int i)
{
  if (i & !FLAGS) // warning
  {
    // code
  }
}
```

 이 경고를 해결 하려면 다음 코드를 사용 합니다.

```
#define FLAGS   0x4004

void f(int i)
{
  if (i & ~FLAGS)
  {
    // code
  }
}
```
