# <a name="c28208"></a>C28208
경고 C28208: 함수 \<function > \<file > (\<line >)에서 다른 매개 변수 목록을 사용 하 여 이전에 정의 되었습니다. 일부 분석 도구는 잘못 된 결과를 생성 합니다.

 이 경고는 함수의 알려진 정의가 다른 항목과 일치 하지 않을 때 보고 됩니다.
