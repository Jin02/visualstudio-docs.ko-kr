# <a name="enterprise-mobility--security-ems-in-visual-studio-subscriptions"></a>Visual Studio의 EMS(Enterprise Mobility + Security)

이전에 선택된 Visual Studio 구독에서 독립적으로 사용할 수 있었던 Enterprise Mobility + Security는 이제 동일한 선택된 Visual Studio 구독에 포함된 Microsoft 365 개발자 구독에 포함됩니다.  

따라서 Enterprise Mobility + Security는 더 이상 별도로 제공되지 않습니다.  

Microsoft 365 개발자 구독을 활성화하려면:
1. [Visual Studio 구독 포털](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs)에 로그인합니다.
1. Microsoft 365 개발자 구독 타일을 찾고 **활성화**를 클릭합니다.  
1. 프롬프트에 따라 계정 만들기를 완료합니다.  

[Microsoft 365 개발자 구독](vs-m365.md)을 활성화하는 방법을 자세히 알아봅니다. 

## <a name="support-resources"></a>지원 리소스
- EMS와 관련하여 도움이 필요하세요?  온라인 [Enterprise Mobility + Security 설명서](/enterprise-mobility-security/)를 확인하세요.
- EMS의 구성 요소를 사용하는 기술 지원은 [Enterprise Mobility + Security 지원 FAQ](https://docs.microsoft.com/enterprise-mobility-security/solutions/ems-support-faq)를 방문하여 질문에 대한 적합한 지원을 찾을 수 있습니다.
- Visual Studio 구독에 대한 판매, 구독, 계정 및 요금 청구에 대한 지원을 받으려면 Visual Studio [구독 지원](https://visualstudio.microsoft.com/subscriptions/support/)에 문의하세요.
- Visual Studio IDE, Azure DevOps Services 또는 기타 Visual Studio 제품이나 서비스와 관련하여 궁금한 점이 있나요?  [Visual Studio 지원](https://visualstudio.microsoft.com/support/)을 참조하세요.
