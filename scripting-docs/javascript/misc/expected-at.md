# <a name="expected-"></a>' @No__t_0 '가 필요 합니다.
@No__t_0 문을 사용 하 여 조건부 컴파일 문에서 사용할 변수를 만들려고 했지만 변수 이름 앞에 " **@** " 기호를 넣지 않았습니다.  
  
### <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 변수 이름 바로 앞에 " **@** " 기호를 추가 합니다. 예를 들면,  
  
    ```JavaScript  
    @set @myvar = 1  
    ```
