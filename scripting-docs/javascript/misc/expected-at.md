# <a name="expected-"></a>예상 '\@'
사용 하 여 조건부 컴파일 문에서 사용할 변수를 만들려고 합니다 `@set` 문을 배치 하지 않고 있지만 at 기호 "**@**" 변수 이름 앞입니다.  
  
### <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   추가 at 기호 "**@**" 변수 이름 바로 앞입니다. 예를 들어:  
  
    ```JavaScript  
    @set @myvar = 1  
    ```
