# <a name="expected-"></a>'{'가 필요합니다.
함수 본문의 시작을 표시 하는 왼쪽된 중괄호를 포함 되지 않았습니다. 코드 줄 이더라도 함수의 본문을 구성 하는 왼쪽 및 오른쪽 중괄호 안에 포함 되어야 합니다. 함수 본문에는 중괄호를 사용 하 여 루프의 보다 엄격한 인지 확인 합니다.  
  
### <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 함수 본문의 시작을 표시 하는 왼쪽된 중괄호를 추가 합니다.
