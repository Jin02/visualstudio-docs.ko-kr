# <a name="javascript-object-expected"></a>JavaScript 개체가 필요합니다.
[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 개체가 필요한 기본 제공 함수에[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] 아닌 개체를 전달 하려고 했습니다. 다양 한 기본 제공 함수에는 호스트에서 정의 된 개체 또는 컨트롤과 같은 외부 구성 요소와는 달리 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]에 정의 된 개체가 필요 합니다.  
  
### <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 매개 변수로 전달 하는 개체의 형식이 올바른지 확인 합니다.
