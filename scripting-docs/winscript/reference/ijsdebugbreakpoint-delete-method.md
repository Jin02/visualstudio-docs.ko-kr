# <a name="ijsdebugbreakpointdelete-method"></a>IJsDebugBreakPoint::Delete 메서드
중단점을 삭제합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp
HRESULT Delete(void);  
```  
  
## <a name="return-value"></a>반환 값  
  
## <a name="remarks"></a>주의  
 삭제 된 중단점에서 호출 되는 경우 S_FALSE를 반환 합니다.
