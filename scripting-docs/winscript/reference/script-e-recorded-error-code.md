# <a name="script_e_recorded-error-code"></a>SCRIPT_E_RECORDED 오류 코드
스크립트 엔진과 호스트 간에 오류가 전달 되었습니다. 호스트에서 호출자에 게 오류 코드를 전달 해야 합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
SCRIPT_E_RECORDED   0x86664004L  
```
