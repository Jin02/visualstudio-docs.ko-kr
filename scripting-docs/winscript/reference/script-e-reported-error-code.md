# <a name="script_e_reported-error-code"></a>SCRIPT_E_REPORTED 오류 코드
스크립팅 엔진이 [IActiveScriptSite:: OnScriptError](../../winscript/reference/iactivescriptsite-onscripterror.md)를 통해 호스트에 처리 되지 않은 예외를 보고 했습니다. 호스트에서이 오류를 무시할 수 있습니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
SCRIPT_E_REPORTED   0x80020101L  
```
