# <a name="idebugdocumenthelpersettextattributes"></a>IDebugDocumentHelper::SetTextAttributes
텍스트 범위에 대 한 특성을 설정 하 고 해당 텍스트의 다른 특성을 재정의 합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp
HRESULT SetTextAttributes(  
   ULONG              ulCharOffset,  
   ULONG              cChars,  
   SOURCE_TEXT_ATTR*  pstaTextAttr  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `ulCharOffset`  
 진행 텍스트 범위의 시작 위치입니다.  
  
 `cChars`  
 진행 범위의 문자 수입니다.  
  
 `pstaTextAttr`  
 진행 텍스트 범위에 대 한 소스 텍스트 특성입니다.  
  
## <a name="return-value"></a>반환 값  
 이 메서드는 `HRESULT`를 반환합니다. 가능한 값에는 다음 표에 있는 값이 포함되지만, 이에 국한되는 것은 아닙니다.  
  
|값|설명|  
|-----------|-----------------|  
|`S_OK`|메서드가 성공했으며|  
  
## <a name="remarks"></a>주의  
 텍스트를 문서에 추가 하기 전에 텍스트 범위에서 `SetTextAttributes`를 호출 하는 것은 오류입니다. `AddDBCSText`, `AddUnicodeText`또는 `AddDeferredText` 메서드를 호출 하 여 문서에 텍스트를 추가 합니다.
