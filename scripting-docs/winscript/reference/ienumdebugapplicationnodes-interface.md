# <a name="ienumdebugapplicationnodes-interface"></a>IEnumDebugApplicationNodes 인터페이스
애플리케이션과 연결된 노드의 자식 노드를 열거합니다.  
  
 상속 된 메서드 외에도 `IUnknown`, `IEnumDebugApplicationNodes` 인터페이스는 다음 메서드를 노출 합니다.  
  
## <a name="methods-in-vtable-order"></a>Vtable 순서의 메서드  
  
|메서드|설명|  
|------------|-----------------|  
|[IEnumDebugApplicationNodes::Next](../../winscript/reference/ienumdebugapplicationnodes-next.md)|열거형 시퀀스에서 세그먼트의 지정된 된 수를 검색 합니다.|  
|[IEnumDebugApplicationNodes::Skip](../../winscript/reference/ienumdebugapplicationnodes-skip.md)|열거형 시퀀스에서 세그먼트의 지정 된 수를 건너뜁니다.|  
|[IEnumDebugApplicationNodes::Reset](../../winscript/reference/ienumdebugapplicationnodes-reset.md)|열거형 시퀀스를 처음으로 다시 설정합니다.|  
|[IEnumDebugApplicationNodes::Clone](../../winscript/reference/ienumdebugapplicationnodes-clone.md)|현재 열거자와 동일한 상태를 포함 하는 열거자를 만듭니다.|
