# <a name="script_e_propagate-error-code"></a>SCRIPT_E_PROPAGATE 오류 코드
스크립트 오류가 호출자에 게 전파 되는 중입니다 .이는 다른 스레드에 있을 수 있습니다. 호스트에서 호출자에 게 오류 코드를 전달 해야 합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
SCRIPT_E_PROPAGATE  0x80020102L  
```
