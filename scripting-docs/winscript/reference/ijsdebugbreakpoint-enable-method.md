# <a name="ijsdebugbreakpointenable-method"></a>IJsDebugBreakPoint::Enable 메서드
중단점을 설정 합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp
HRESULT Enable(void);  
```  
  
## <a name="return-value"></a>반환 값  
  
## <a name="remarks"></a>주의  
 삭제 된 중단점에서 호출 된 경우 E_UNEXPECTED를 반환 합니다.  
  
## <a name="requirements"></a>요구 사항  
 **헤더:** jscript9diag.h
